Projects

File Device Driver

     In university, I wrote a file device driver which can execute open, close, read, write operations of a file. It supports multiple open files simultaneously and uses a LRU cache for efficiency. Moreover, it supports network communication to a server that implements the memory device.


Single-Cycle CPU

    I designed a single-cycle CPU using the Verilog language in Vivado environment. It implements MIPS five stages, which are fetch the instructions from the instruction memory, decode the instructions, execute the instructions and write/read the memory. The program is implemented onto the Zybo board, and the LED will light up according to corresponding instructions.


Database Application

    Our team designed a database application to manage the online business of a startup company. It combines the functions of Amazon and eBay. In our database design, there are fixed price items and auction ones, and sellers and users. It implements functions like selling items, searching categories, registering, rating, bidding, ordering, writing sale reports, and deliverying etc.




Making an Aircraft Game

     Using C++ language and Cocos 2d game engine to make an aircraft war game. It includes basic functions like switching scene, randomly creating enemy, scoring, collision detection of bullet and enemy, user and enemy, and death detection. User can also select different difficulties to play.


Hitting Brick Small Game

     Wrote a hitting brick small game using cocos2d game engine in C++ language. User need to catch and emit the ball to destroy all tiles in the screen through moving the bottom board. There are different tiles which need to be hit one, two, or three times. There are also different kinds of items to be picked up which can let the ball go through everything, add one more life, double or triple the ball to make game easier.
